Create a high-authority, heavy, masculine roofing contractor website.

The design must feel:
- Established
- Industrial
- Structurally strong
- Confident
- Premium
- Solid
- Impactful
- Revenue-driven

This is NOT a startup website.
This is NOT minimal tech SaaS.
This is NOT soft or rounded.
This is NOT playful.
This is NOT smooth or bubbly.

The site must feel like:
A $10M+ revenue roofing company dominating its city.

----------------------------------
GLOBAL THEME SYSTEM
----------------------------------

GRID SYSTEM:
- 1440px max width container
- 12-column grid
- 80px outer margin left and right
- 24px gutter spacing between columns
- Sections vertically spaced with minimum 120px padding top and bottom
- Large breathing space between sections (140–160px)

BACKGROUND STRATEGY:
- Primary background: Pure White (#FFFFFF)
- Section contrast backgrounds: Very light gray (#F4F5F6)
- Dark CTA sections: Deep Charcoal (#111111)
- No gradients.
- No glassmorphism.
- No blur backgrounds.

COLOR SYSTEM:
Primary: Deep Navy (#0D1B2A)
Secondary: Charcoal (#1B263B)
Accent: Roofing Red (#B11226)
Neutral Dark: #111111
Light Neutral: #F4F5F6
Muted Text: #6B7280

Buttons:
Primary button: Solid Roofing Red background with white text.
Hover: Darker red (#8E0F1F)
Secondary button: Transparent with 2px solid Navy border.

TYPOGRAPHY:
Headlines:
- Bold condensed sans-serif or strong serif
- Weight: 700–800
- Tight tracking
- Uppercase for key headings

Body:
- Clean sans-serif
- 18px default body size
- Line height 1.6

H1:
- 64–72px
- Line height 1.1
- Max width 800px

H2:
- 42–48px
- Line height 1.2

H3:
- 26–30px
- Line height 1.3

Button Text:
- 18px bold uppercase
